# ASC_Winter_Training_Camp
## 仓库文件结构:
$$GIT仓库\left\{
\begin{aligned}
docs 存放文档文件\\
Firmware放置项目代码\\
Hardware放置PCB文件\\
Mechanical放置各类3D文件
\end{aligned}
\right.
$$
## 项目代码结构
$$项目代码\left\{
\begin{aligned}
空项目自带文件\\
App文件夹用于存放算法\\
Bsp用于底层驱动\\
Config存放系统设置项与小车数据
\end{aligned}
\right.$$
